# 🌤️ SkyTrack

Aplicação web desenvolvida em React.js para consulta de clima em tempo real utilizando a API Open-Meteo.

Este projeto foi desenvolvido como trabalho acadêmico da disciplina **ES47B-ES71 – Programação Web Fullstack**, sob orientação do **Prof. Willian Massami Watanabe**, pelo aluno **Kauan Santos Pedreira - RA:2565900**.

## 📌 Funcionalidades

- Busca de clima por cidade  
- Exibição de temperatura e vento em tempo real  
- Histórico de consultas  
- Persistência de dados no `localStorage`  
- Validação de formulário com React Hook Form  

---

## 🧠 Tecnologias utilizadas

- React.js  
- Vite  
- Context API  
- useReducer (Hooks)  
- React Hook Form  
- Open-Meteo API  
- JavaScript (ES6+)  

---

## 📡 API utilizada

- Open-Meteo (clima e geolocalização)

---

## 💾 Persistência de dados

O histórico de buscas é armazenado no `localStorage` do navegador.
